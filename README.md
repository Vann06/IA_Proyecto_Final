# Robot Médico IA

Proyecto final de CC3045 - Inteligencia Artificial.

## Descripción
Este proyecto compara dos enfoques para la toma de decisiones de un robot médico en un hospital estocástico:

- Value Iteration
- Q-Learning

Ambos métodos se evaluarán en el mismo entorno, con los mismos estados, acciones y recompensas, para comparar su desempeño en métricas como:

- tiempo de entrega
- consumo de batería
- porcentaje de entregas exitosas
- recompensa acumulada

## Objetivo
Analizar cuál de los dos enfoques ofrece mejor desempeño en un entorno hospitalario discreto modelado como GridWorld.
